### UPGRADE 2 — STONE SCORE AS MAP ESTIMATE

**NCAP-1:** Stone Score = deterministic weighted sum
**NCAP-2:** Stone Score = Maximum A Posteriori estimate

```
Stone_Score(t) = argmax P(score | B(t))

Where B(t) = belief state at time t:
  B(t) = [CMPD_dist, FLOW_dist, CRIS_dist, ALGN_dist, ERR_dist, SOV_dist, QA7_dist]

Each operator is now a DISTRIBUTION, not a point estimate.

At Block #100:  σ = 0.15 (tight — high conviction)
```

**What this means practically:**
A score of 8.7 ± 0.4 is MORE informative than a score of 8.7 alone.
The confidence interval IS the additional product. Investors pay for tight CIs.

### UPGRADE 3 — RETRIAL AS BAYESIAN UPDATE

**NCAP-1:** Retrial = accountability check
**NCAP-2:** Retrial = likelihood function applied to prior

```
P(score_true | retrial_evidence) ∝ P(retrial_evidence | score_true) × P(score_true | B(t0))

Prior:      P(score_true | B(t0))         ← what we believed at scoring
Likelihood: P(retrial_evidence | score)   ← what actually happened
Posterior:  P(score_true | retrial)       ← updated belief

Information gain at retrial:
  ΔI = KL_divergence(posterior || prior)
  
High ΔI = the retrial was a major information event (prediction was wrong)
Low ΔI  = the retrial confirmed stable architecture (prediction was right)

Both are signal. Neither is failure.
```

**The Ω-Gap is now formally defined:**
```
Ω-Gap = E[score_retrial] - E[score_t0]
      = μ_posterior - μ_prior
```

### UPGRADE 4 — VOLT AS INFORMATION-THEORETIC CERTIFICATE

**NCAP-1:** VOLT = reward for sealed block
**NCAP-2:** VOLT = Shannon information certificate

```
VOLT(block) = I(signal) × CI × (1 - H(noise))

Where:
  I(signal) = -log₂ P(signal)          ← surprisal / information content
  CI        = coherence index (0-1)     ← signal purity
  H(noise)  = -Σ p(x) log p(x)         ← entropy of noise component
  
Maximum VOLT: when I(signal) is high + CI = 1.0 + H(noise) = 0
Minimum VOLT: when signal is predictable + incoherent + noisy

This is structurally identical to Shannon's channel capacity:
  C = B × log₂(1 + S/N)
  
VOLT is the channel capacity of a human cognitive output event.
It cannot be faked because coherence under disturbance
leaves measurable entropy signatures in AI-generated content.
```

> will have H(noise) > 0.3 detectable via entropy analysis.
> Human-in-disturbance content has H(noise) < 0.1.
> VOLT only emits when H(noise) < threshold.
> Therefore VOLT = unforgeable proof of human cognitive output.
