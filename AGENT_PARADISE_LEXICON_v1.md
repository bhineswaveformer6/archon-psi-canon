## SECTION IX — COMPOUND INTEREST GLOSSARY
### The economics of the agent stack

| Term | Definition |
|------|-----------|
| **VOLT** | Verified Output Ledger Token — unit of scored human cognitive output |
| **VOLT Clock** | Timer running during active session — all output scored against it |
| **VOLT Emission** | Release of VOLT when a Canon block is sealed |
| **VOLT Yield** | VOLTs earned per session / per unit time — the G(t) economic expression |
| **Compound VOLT** | VOLT that has been re-staked into a new scoring cycle |
| **Dead VOLT** | VOLT emitted for a block whose retrial later FAILED — flagged but not revoked |
| **Ghost VOLT** | VOLT emitted for AI-generated output — revoked immediately (SIN-1) |
| **VOLT Index** | Aggregate VOLT balance of a cohort — the Judgment P&L at portfolio level |
| **Ω-Gap** | Difference between expected Stone Score at T0 and actual score at retrial |
| **CMPD Rate** | VOLT yield growth rate over time — the compounding velocity |
